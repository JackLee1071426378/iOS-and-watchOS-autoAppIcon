# iOS and watchOS autoAppIcon
Auto export iOS and watchOS App Icon. 自动生成iOS和watchOS应用图标表。

# Dependencies

# Usage
iOS icon

```
$ icon-ios.py <Picture.png>
```

watchOS icon

```
$ icon-watch.py <Picture.png>
```
